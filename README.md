### 概览
Oops Framework 是一款基于 Cocos Creator 3.x 开发的游戏框架。自 2021 年 7 月开源起，它始终与 Cocos Creator 3.x 的各个版本保持同步更新。与此同时，该框架也在持续迭代版本，不断提升稳定性与易用性。

### 设计理念
Oops Framework 致力于提供一个拥有游戏开发常用技术的框架，从而提高开发效率、降低开发门槛、专注于内容设计、促进技术共享和交流。

### 框架特点
- 提供较为全面的游戏开发常用技术
- 提供可提高开发效率的插件工具
- 提供教学项目以及不同类型的解决方案项目
- 拥有稳定可靠的框架技术
- 对新人极为友好，操作简单且易于上手
